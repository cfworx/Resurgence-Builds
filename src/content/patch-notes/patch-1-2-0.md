## Patch 1.2.0 — Season 1 Launch

**Release Date:** April 1, 2026
**Maintenance Window:** 3:00 AM – 7:00 AM EDT
**Download Size:** 4.2 GB (mobile), 8.1 GB (PC/console)

## New Content

### Vanguard Specialization
- New frontline-focused specialization added
- Unique skill tree with 24 passive nodes
- Signature skill: Bulwark Shield (enhanced variant with melee combo)
- Two new skill mod combo sets: Spearhead and Blitz Tactics

### Gear Sets
- **Heartbreaker** — Body shot DPS set with stacking armor/damage
- **Umbra Initiative** — Team-oriented set creating weakness zones
- **Rigger** — Skill destruction/redeployment set

### Strongholds
- Capitol Building (4-player, all difficulties)
- Roosevelt Island (4-player, all difficulties)

## Weapon Balance

### Assault Rifles
- Police M4: No changes (already well-positioned)
- AK-47: Stability increased by 5%
- FAMAS: Recoil pattern adjusted, slightly more vertical
- G36: Base damage increased by 3%

### SMGs
- Vector SBR: No changes
- MP5: Optimal range increased by 2 meters
- UMP-45: Rate of fire increased by 4%
- AUG: Base damage reduced by 3% (rebalance pending)

### LMGs
- RPK-74: Base damage increased by 8%, magazine size reduced from 75 to 65
- L86 LSW: Handling improved across all stats by 5%
- M60: Suppression effect duration increased by 2 seconds

### Marksman Rifles
- M700 Carbon: Headshot damage multiplier increased by 5%
- Hunting M44: No changes
- SRS: Base damage increased by 4%

## Bug Fixes

- Fixed an issue where Foundry Bulwark 4-piece bonus armor could exceed the intended cap
- Fixed a crash that occurred when deploying two skills simultaneously in PvP
- Fixed incorrect damage numbers displaying for Measured talent in the second half of the magazine
- Fixed a UI issue where OS Protocol stats would not update after recalibration
- Fixed an exploit allowing infinite skill cooldown reduction with Hard Wired and specific timing
- Fixed several map geometry issues in the Financial District dark zone

## Known Issues

- Rigger 4-piece may occasionally not return the skill if the destruction animation is interrupted
- Heartbreaker stacks can display incorrectly on the UI (actual values are correct server-side)
- Capitol Building boss #2 may occasionally reset if all players die simultaneously

## Developer Notes
